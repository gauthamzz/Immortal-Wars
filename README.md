# Prequisites

1. freeglut or OpenGL Library `sudo apt-get install freeglut3`
2. C++ `sudo apt-get install g++'

# To run

Linux Users

1. `g++ main.cpp -o gl -lGL -lGLU -lglut` 
2. `./gl`
